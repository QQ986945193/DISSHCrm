# DISSHCrm

这是一个客户管理系统。CRM架构采用struts2,Spring,Hibernate来完成的。

当然还没有全部完善，功能正在不断完善更新中。。。

- **开发工具：IntelliJ IDEA**   

- **jdk版本：1.8** 

- **Apache Tomcat/8.5.12** 

- **数据库：mysql**
